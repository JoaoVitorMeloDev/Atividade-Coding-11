# 🔭 Observatório de Projetos

Projeto web simples desenvolvido com HTML, CSS e JavaScript para demonstrar interação com o DOM, manipulação de elementos e estilização moderna de interface.

---

## 📌 Sobre o Projeto

O sistema permite que o usuário digite o nome de um projeto integrador e visualize uma mensagem dinâmica na tela.

Além disso, o projeto possui:

- Interface moderna em formato de card
- Botões interativos
- Manipulação de elementos HTML com JavaScript
- Exibição dinâmica de imagem
- Estilização responsiva e organizada

---

## 🚀 Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript

---

## 📂 Estrutura do Projeto

```bash
frontend/
│
├── index.html
├── style.css
└── script.js
```

---

## 🖥️ Funcionalidades

- Inserir nome de projeto
- Exibir mensagem personalizada
- Validar campo vazio
- Alterar cores dinamicamente
- Mostrar e ocultar imagem
- Limpar informações da tela

## ▶️ Como Executar

### 1. Clone o repositório

```bash
git clone https://github.com/SEU-USUARIO/SEU-REPOSITORIO.git
```

### 2. Acesse a pasta

```bash
cd SEU-REPOSITORIO
```

### 3. Execute o projeto

Basta abrir o arquivo:

```bash
index.html
```

no navegador.
